# Linear Regression Learning Resources

## 📚 Contents

### Notebooks:

1. **CampusX session.ipynb**
   - Custom Linear Regression implementation from scratch
   - Manual calculation of slope (m) and intercept (b)
   - Uses CGPA dataset for placement prediction

2. **Linear Regression.ipynb**
   - Basic Linear Regression with sklearn
   - Simple house price prediction example
   - Quick introduction to model.fit() and model.predict()

3. **Practical Coding Practice - LINEAR REGRESSION.ipynb**
   - Comprehensive practice notebook
   - Synthetic dataset creation with noise
   - Train-test split (80-20)
   - Model training and evaluation
   - Metrics: MAE, MSE, RMSE, R²
   - Feature scaling comparison (StandardScaler & MinMaxScaler)
   - Residual analysis and visualization
   - Regression line plots

### Dataset:
- **cgpa_data.csv** - CGPA vs Placement Package (used in CampusX notebook)
  - 96 rows of data
  - Columns: CGPA, Package_LPA
